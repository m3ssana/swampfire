Update TODO.md and related GitHub issues after completing the current task.

Work through each step in order:

1. **Update TODO.md** — Read TODO.md and mark any items completed by this task. Add follow-up tasks if any were discovered. Edit the file if changes are needed.

2. **Update/close related GitHub issues** — Identify which GitHub issues this task addressed (from branch name, commit messages, or context). Use `gh issue comment` to summarize what was done, then close with `gh issue close` if fully resolved.

After each step, report what was done or confirm it was already clean. If a step requires user input (e.g. which issue to close), ask before proceeding.
